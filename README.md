# YouTube Music Discord RPC extension for Firefox

Same as the Spotify status but for YouTube Music.

> [!CAUTION]
> Only supports **Firefox**! Pls tell me if you want Chrome support.

## Usage

Start the host application. It automatically connects to your Discord client (if it's running). You can make it automatically start on system boot.

The extension automatically connects to the host (if it's running).

### Extension icon meanings

- ![Plain icon](https://raw.githubusercontent.com/BBpezsgo/YouTube-Music-Discord-RPC/refs/heads/main/extension/icons/icon16.png) - No music is playing (no status is shown)
- ![Icon with red exclamation mark](https://raw.githubusercontent.com/BBpezsgo/YouTube-Music-Discord-RPC/refs/heads/main/extension/icons/icon16-off.png) - Couldn't connect to the host
- ![Icon with light blue hourglass](https://raw.githubusercontent.com/BBpezsgo/YouTube-Music-Discord-RPC/refs/heads/main/extension/icons/icon16-loading.png) - Currently connecting to the host
- ![Icon with green checkmark](https://raw.githubusercontent.com/BBpezsgo/YouTube-Music-Discord-RPC/refs/heads/main/extension/icons/icon16-on.png) - Connected to the host and is showing a status on Discord
