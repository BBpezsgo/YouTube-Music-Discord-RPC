# YouTube Music Discord RPC extension for Firefox

Same as the Spotify status but for YouTube Music.

## Usage

Start the host application. It automatically connects to your Discord client (if it's running). You can make it automatically start on system boot.

The extension automatically connects to the host (if it's running).

### Extension icon meanings

- ![Plain icon](https://raw.githubusercontent.com/BBpezsgo/YouTube-Music-Discord-RPC/refs/heads/main/extension/icons/icon16.png) - No music is playing so no status is shown
- ![Icon with red exclamation mark](https://raw.githubusercontent.com/BBpezsgo/YouTube-Music-Discord-RPC/refs/heads/main/extension/icons/icon16-off.png) - Extension couldn't connect to the host
- ![Icon with green checkmark](https://raw.githubusercontent.com/BBpezsgo/YouTube-Music-Discord-RPC/refs/heads/main/extension/icons/icon16-on.png) - Extension is connected to the host and is showing a status on Discord
